# API Inquiry v0.2.0

## 新增功能

- 新增本地 API Inquiry 控制台窗口。
- 将 DeepSeek API Key 配置、更换和删除从菜单栏面板迁移到控制台中。
- 支持导入 DeepSeek Usage 官方 zip 导出包，并保留兼容 CSV 兜底。
- 在本地展示用量总览、模型汇总和明细记录。
- 将归一化后的用量记录保存到本机 Application Support，不保存原始导出文件。
- 菜单栏面板保持轻量，只保留余额状态、刷新、打开控制台、AutoStart 和退出。

## 验证下载文件

```bash
shasum -a 256 -c API-Inquiry-v0.2.0.dmg.sha256
```

## 已知限制

- 未进行 Apple notarization 公证。
- 暂无用量趋势图。
- 用量导入采用替换当前本地数据集的策略，暂不合并多个历史文件。
- 当前界面只显示 DeepSeek 一个供应商。
