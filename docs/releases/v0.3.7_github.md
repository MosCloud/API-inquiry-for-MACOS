# API Inquiry v0.3.7

## 中文

### 应用优化

- 刷新反馈更稳了：API Inquiry 正在检查供应商时，刷新图标会以克制的节奏向前转动，刷新结束后自然停下。
- 余额、用量和额度数字变化更顺滑；在较新的 macOS 上更明显，macOS 13 仍保持正常兼容。
- 保存、删除、警告和错误提示更容易一眼识别，状态图标很轻，不会打断当前操作。
- warning 颜色更柔和，并在浅色和深色模式下保持一致，低余额或高用量状态更醒目但不刺眼。
- 菜单栏状态、额度窗口、供应商行、状态 badge 和图标按钮补充了更清楚的辅助功能描述。
- Console Home 和 Console API 保持原来的简洁布局，但关键状态和操作更容易扫读。

### 问题修复

- 修复语言切换后 API 接入状态颜色可能不准确的问题。
- 修复刷新动画接近一圈结束时可能出现停顿或回退感的问题。
- 当 macOS 开启“减少动态效果”时，本次新增动效会降级或关闭。
- Console Home 的供应商行不再影响内部按钮被辅助功能单独访问。

### 下载校验

```bash
shasum -a 256 -c API-Inquiry-v0.3.7.dmg.sha256
```

### 已知限制

- Apple notarization 尚未启用。
- 本版本聚焦 UI 优化，不新增供应商。

## English

### App Optimization

- Refresh now feels steadier: the icon turns forward at a calm pace while API Inquiry is checking providers, then stops naturally when the refresh finishes.
- Balance, usage, and quota numbers change more smoothly on newer macOS versions while macOS 13 remains fully supported.
- Save, delete, warning, and error messages are easier to notice at a glance, with small status icons that stay quiet and do not interrupt your workflow.
- Warning colors are softer and more consistent in light and dark mode, so low-balance or high-usage states stand out without making the interface noisy.
- Menu bar status, quota windows, provider rows, status badges, and icon buttons now have clearer accessibility descriptions.
- Console Home and Console API keep the same simple layout, but key states and actions are easier to scan.

### Bug Fixes

- Fixed an issue where API access status could be colored incorrectly after language changes.
- Fixed the refresh animation so it no longer appears to pause or jump backward near the end of a turn.
- When Reduce Motion is enabled in macOS, new motion effects are reduced or removed.
- Console Home provider rows now keep their internal buttons available to assistive technologies.

### Verify Downloads

```bash
shasum -a 256 -c API-Inquiry-v0.3.7.dmg.sha256
```

### Known Limitations

- Apple notarization is not enabled.
- This release focuses on UI polish and does not add new providers.
