# API Inquiry v0.1.1-alpha.1

API Inquiry v0.1.1-alpha.1 是 DeepSeek 余额菜单栏应用的体验打磨版本。

## 主要变化

- 优化首次启动时的 DeepSeek API Key 配置引导。
- 刷新失败时提供可操作的恢复动作，例如重试或更换 key。
- API Key 管理增加删除确认和保存反馈。
- 刷新按钮会体现正在刷新状态。
- 提供版本化 DMG 文件和 SHA-256 checksum。

## 下载文件

- `API-Inquiry-v0.1.1-alpha.1.dmg`
- `API-Inquiry-v0.1.1-alpha.1.dmg.sha256`

## 校验下载文件

```bash
shasum -a 256 -c API-Inquiry-v0.1.1-alpha.1.dmg.sha256
```

## 安装

1. 下载 DMG。
2. 打开 DMG。
3. 将 `API Inquiry.app` 拖入 `Applications`。
4. 从 Applications 启动 API Inquiry。

应用使用 ad-hoc 签名，但没有 Apple notarization 公证。首次启动时，macOS 可能需要你右键点击 app 并选择 `Open`，或在 `System Settings > Privacy & Security` 中允许打开。

## 已知限制

- 未进行 Apple notarization 公证。
- 没有详细用量图表。
- 暂无本地 DeepSeek 用量控制台。
- 当前 alpha 界面只显示 DeepSeek 一个供应商。
