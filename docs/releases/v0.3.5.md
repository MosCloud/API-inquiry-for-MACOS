# API Inquiry v0.3.5

## Polish

- Refined the Console Home page so provider rows better match the translucent visual language of the menu details panel.
- Added health-tinted provider row backgrounds with a softer left-to-right gradient wash that fades early and keeps the main content readable.
- Changed Home page resource badges to clearer balance/quota language: sufficient, low, and critical states now distinguish balance-style providers from quota-style providers.
- Aggregated quota health now uses the most urgent state across 5h and weekly remaining windows, so the Home page reflects the tighter limit first.
- Aligned Console module spacing, provider module height, translucency, and section title placement across Home, API, and Settings.
- Bumped the app metadata and release packaging configuration to v0.3.5.

## Fixes

- Fixed Console Home and API pages clipping provider modules at the bottom of the window by making the page content area vertically scrollable while keeping the navigation and section title anchored.
- Fixed OpenAI/Codex-style providers without API key management showing key controls in the API page; these providers now only show provider removal.
- Fixed Console navigation and section title positions drifting on Settings because the short Settings page did not fill the host window.
- Fixed Settings showing the previous v0.3.4 version number by synchronizing the release version source.
