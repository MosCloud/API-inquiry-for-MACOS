# API Inquiry v0.3.5

## 优化

- 优化 Console Home 页，让供应商模块更贴近菜单栏详情页的半透明视觉语言。
- 为供应商模块增加健康度染色背景，并将左到右渐变调得更柔和、更早淡出，减少对主体内容的干扰。
- 调整 Home 页右上角资源标签文案：余额类供应商使用余额充足、余额偏低、余额告急；额度类供应商使用额度充足、额度偏低、额度告急。
- Home 页额度健康度现在会聚合 5h remaining 和 weekly remaining，优先展示更紧急的状态。
- 统一 Console Home、API、Settings 三个页面的模块间距、供应商模块高度、透明质感和页面标题位置。
- 将应用元数据和发布打包配置同步到 v0.3.5。

## 问题修复

- 修复 Console Home 和 API 页面在窗口底部裁切供应商模块的问题，现在页面内容区支持上下滚动，同时保持导航栏和页面标题位置固定。
- 修复 OpenAI/Codex 这类暂不支持 API Key 管理的供应商在 API 页面错误展示密钥管理控件的问题，现在只展示移除供应商入口。
- 修复 Settings 页面因为内容较短导致 Console 导航和页面标题位置下移的问题。
- 修复 Settings 中版本号仍显示 v0.3.4 的问题。
