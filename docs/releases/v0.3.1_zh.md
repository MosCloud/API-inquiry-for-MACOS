# API Inquiry v0.3.1

## 新增功能

- 新增 OpenAI/Codex provider，支持通过 ChatGPT/Codex 会话型额度接口查询 Codex quota。
- OpenAI/Codex 会优先自动读取本机 Codex 登录态（`$CODEX_HOME/auth.json` 或 `~/.codex/auth.json`），无需在 Console 手工输入 OpenAI Platform API key。
- OpenAI/Codex 作为主菜单项时，菜单栏沿用紧凑图片标签设计，显示 OpenAI 图标和 `5h xx%` 剩余额度。
- OpenAI/Codex 详情页展示 `5h` 和 `7d` 两个剩余额度窗口；`5h` reset 显示时间，`7d` reset 显示日期。
- OpenAI/Codex 作为从单元时显示为 OpenAI，并按两行展示 `5h xx%` 和 `7d xx%`。
- Console Home 展示 OpenAI/Codex 当前 plan，例如 Free、Plus、Pro 5x、Pro 20x、Business 或 Enterprise/Edu。
- Console Home 的 Detail 明确标注语义：智谱显示 `used`，OpenAI/Codex 显示 `remg`。

## UI 优化

- 新增 OpenAI header/logo 和 16x16 menu icon，详情页和 Console 统一展示 OpenAI 品牌。
- 详情页移除从单元的 `Last updated`，只保留主展示单元的更新时间。
- 智谱作为从单元时，将 `5h xx%` 和 `Resets: xx:xx` 放在同一行展示。

## 已知限制

- 未进行 Apple notarization 公证。
- OpenAI/Codex 额度查询依赖 ChatGPT/Codex 会话型 backend，接口可能随 Codex 官方实现变化。
- 当前只支持内置供应商，不支持任意自定义 endpoint。
