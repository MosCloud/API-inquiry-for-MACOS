# API Inquiry v0.3.1

## 新增功能

- 新增 Codex provider，支持通过 ChatGPT/Codex 会话型额度接口查询 Codex quota。
- Codex 会优先自动读取本机 Codex 登录态（`$CODEX_HOME/auth.json` 或 `~/.codex/auth.json`），无需在 Console 手工输入 OpenAI Platform API key。
- Codex 作为 Primary Provider 时，菜单栏沿用紧凑图片标签设计，显示 ChatGPT/GPT 标识和 `5h xx%` 剩余额度。
- Codex 详情页展示 5h 和 Week 两个剩余额度窗口，以及对应 reset 时间。
- Console Home 展示 Codex 当前 plan，例如 Free、Plus、Pro 5x、Pro 20x、Business 或 Enterprise/Edu。

## 验证下载文件

```bash
shasum -a 256 -c API-Inquiry-v0.3.1.dmg.sha256
```

## 已知限制

- 未进行 Apple notarization 公证。
- Codex 额度查询依赖 ChatGPT/Codex 会话型 backend，接口可能随 Codex 官方实现变化。
- 当前只支持内置供应商，不支持任意自定义 endpoint。
