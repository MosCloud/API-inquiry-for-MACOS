# API Inquiry v0.3.2

## 新增功能

- 新增中文 UI，本机 macOS 首选语言为中文时默认显示中文。
- 新增语言设置，Console Home 可切换 `自动 / 中文 / English`。
- 语言选择会持久化保存；默认 `自动` 会跟随系统语言。
- 菜单栏详情页和 Console 会在切换语言后同步更新。

## 本地化范围

- 菜单栏详情页、Console Home、Console API、按钮、状态、错误提示、设置反馈、最近更新时间和 reset 文案均支持中英文。
- `API Key` 中文统一为 `API 密钥`。
- `Keychain` 中文统一为 `密钥串`。
- 中文 UI 中 Codex 额度窗口显示为紧凑标签：`5 时` 和 `1 周`。

## 保持原文

- Provider 和产品名保持原文，例如 `OpenAI`、`DeepSeek`、`Zhipu GLM Coding Plan`、`Codex`、`ChatGPT`。
- 通用符号和计划名保持原文，例如 `API`、`CNY`、`Free`、`Plus`、`Pro`、`Prolite`、`Business`。

## 验证下载文件

```bash
shasum -a 256 -c API-Inquiry-v0.3.2.dmg.sha256
```

## 已知限制

- 未进行 Apple notarization 公证。
- 当前只支持中文和英文，不支持更多语言。
- 语言切换不改变 provider 查询逻辑、已保存 API 密钥、Codex 登录态、Primary Provider 或最后一次快照。
