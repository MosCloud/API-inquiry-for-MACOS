# API Inquiry v0.3.3

## 优化

- 菜单详情面板中，主供应商的“可用”/“Available”状态文字现在会与下方供应商状态使用一致的绿色，提升状态信息的视觉一致性。
- Console 首页中，当前显示在菜单栏的供应商标记改为与“开机自启”一致的蓝色高亮效果。
- Console 新增“设置”页面，语言设置从首页迁移到设置页。
- 设置页新增版本信息和“项目主页”按钮，可直接打开 GitHub 项目主页。

## 下载校验

```bash
shasum -a 256 -c API-Inquiry-v0.3.3.dmg.sha256
```
