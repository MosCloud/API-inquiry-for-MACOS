# API Inquiry v0.3.9

## New Features

- Console Home now shows Codex manual reset credits on the OpenAI/Codex provider card, including the available reset-card count and nearest expiration.
- Manual reset credits can be refreshed independently from the Console Home metric without joining the main menu bar refresh chain.

## App Optimization

- Manual reset credit checks use a separate low-frequency cache, so they do not block the regular 5-minute provider refresh or a menu bar manual refresh.
- Menu bar refresh and manual reset refresh now use the same lightweight in-progress, success, and failure feedback pattern.
- The OpenAI/Codex Home card keeps quota, manual reset credits, and last-updated details inside the existing compact provider layout.

## Bug Fixes

- Fixed repeated menu bar refreshes making the refresh icon appear faster over time.

shasum -a 256 -c API-Inquiry-v0.3.9.dmg.sha256

## Known Limitations

- Apple notarization is not enabled.
- Manual reset credits depend on the current local Codex/ChatGPT credentials. If those credentials expire, refresh the login before querying again.
