# API Inquiry v0.3.10

## 中文

### 新增功能

- Console Home 的 OpenAI/Codex 手动重置摘要现在可以打开详情页，展示所有可用重置卡。
- 手动重置详情中的发放时间和过期时间会按当前系统时区显示。

### 应用优化

- 应用内时间展示统一跟随当前系统时区和 12/24 小时制偏好。
- Console Home 删除“更新于”指标列，为额度和手动重置详情留出更稳定的展示空间。
- 手动重置摘要保留可点击胶囊风格，同时与其他 Home 指标保持一致的数值字体。

### 问题修复

- 修复手动重置刷新仍在进行时删除或更换 Codex 凭证后，旧结果回写状态的问题。
- 修复 Codex 凭证变化后手动重置缓存可能复用旧账号结果的问题。
- 修复未配置 Codex 时仍暴露空手动重置详情入口的问题。
- 修复 Codex quota reset 时间以毫秒数字字符串返回时的解析问题。
- 将 Settings 页面和打包元信息同步到 v0.3.10。

### 下载校验

```bash
shasum -a 256 -c API-Inquiry-v0.3.10.dmg.sha256
```

### 已知限制

- Apple notarization 尚未启用。
- 手动重置额度依赖当前本机 Codex/ChatGPT 凭证；如果凭证过期，需要先刷新登录态再查询。

## English

### New Features

- The OpenAI/Codex manual reset summary in Console Home now opens a detail sheet with every available reset card.
- Manual reset details show granted and expiration times in the current system time zone.

### App Optimization

- Time displays now consistently follow the user's current system time zone and 12-hour/24-hour clock preferences.
- Console Home removes the last-updated metric column to keep provider rows compact and give quota/manual-reset details enough room.
- Manual reset summaries keep the clickable pill styling while preserving the same value typography as other Home metrics.

### Bug Fixes

- Fixed stale in-flight manual reset refreshes restoring removed or changed Codex state.
- Fixed manual reset cache reuse across changed Codex credentials.
- Fixed unconfigured Codex rows exposing an empty manual reset detail action.
- Fixed Codex quota reset timestamps that arrive as numeric millisecond strings.
- Synchronized Settings and packaging version metadata to v0.3.10.

### Verify Downloads

```bash
shasum -a 256 -c API-Inquiry-v0.3.10.dmg.sha256
```

### Known Limitations

- Apple notarization is not enabled.
- Manual reset credits depend on the current local Codex/ChatGPT credentials. If those credentials expire, refresh the login before querying again.
