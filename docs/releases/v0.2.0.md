# API Inquiry v0.2.0

## 中文

### 新增功能

- 新增本地 API Inquiry 控制台窗口。
- 将 DeepSeek API Key 配置、更换和删除从菜单栏面板迁移到控制台中。
- 新增轻量 Console Home 页面，展示供应商 API Key 状态、生效状态和余额状态。
- 新增 Console API 页面，用于管理供应商 API Key。
- Console 中的供应商名称可点击；DeepSeek 打开 `https://platform.deepseek.com/usage`。
- 菜单栏面板保持轻量，只保留余额状态、Console 和刷新图标操作、AutoStart 和退出。

### 验证下载文件

```bash
shasum -a 256 -c API-Inquiry-v0.2.0.dmg.sha256
```

### 已知限制

- 未进行 Apple notarization 公证。
- 当前界面只显示 DeepSeek 一个供应商。
- 不包含历史用量导入和用量图表。

## English

### New Features

- Added a local API Inquiry console window.
- Moved DeepSeek API key setup, replacement, and deletion from the menu bar panel into the console.
- Added a lightweight console Home page with provider API key status, validation status, and balance state.
- Added a console API page for provider API key management.
- Made provider names clickable in the console; DeepSeek opens `https://platform.deepseek.com/usage`.
- Kept the menu bar panel lightweight with balance status, Console and refresh icon actions, AutoStart, and quit.

### Verify Download

```bash
shasum -a 256 -c API-Inquiry-v0.2.0.dmg.sha256
```

### Known Limits

- No Apple notarization.
- DeepSeek is the only visible provider in this release.
- Historical usage import and usage charts are not included.

### Todo

- Implement the Add Provider flow in the console.
- Add full multi-provider management after the provider model expands beyond DeepSeek.

- Add historical usage import or charts only if they can stay lightweight and low-friction.
- Add Developer ID signing and Apple notarization before broader distribution.
