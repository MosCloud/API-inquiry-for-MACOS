# API Inquiry v0.2.0

## New Features

- Added a local API Inquiry console window.
- Moved DeepSeek API key setup, replacement, and deletion from the menu bar panel into the console.
- Added DeepSeek Usage CSV import.
- Added local usage totals, model summaries, and detail records.
- Saved normalized usage records to local Application Support without storing the original CSV file.
- Kept the menu bar panel lightweight with balance status, refresh, open console, AutoStart, and quit.

## Verify Download

```bash
shasum -a 256 -c API-Inquiry-v0.2.0.dmg.sha256
```

## Known Limits

- No Apple notarization.
- No usage trend charts yet.
- CSV import replaces the current local dataset and does not merge multiple history files yet.
- DeepSeek is the only visible provider in this release.
