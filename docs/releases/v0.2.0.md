# API Inquiry v0.2.0

## New Features

- Added a local API Inquiry console window.
- Moved DeepSeek API key setup, replacement, and deletion from the menu bar panel into the console.
- Added a lightweight console Home page with provider API key status, validation status, and balance state.
- Added a console API page for provider API key management.
- Made provider names clickable in the console; DeepSeek opens `https://platform.deepseek.com/usage`.
- Kept the menu bar panel lightweight with balance status, Console and refresh icon actions, AutoStart, and quit.

## Verify Download

```bash
shasum -a 256 -c API-Inquiry-v0.2.0.dmg.sha256
```

## Known Limits

- No Apple notarization.
- DeepSeek is the only visible provider in this release.
- Historical usage import and usage charts are not included.
