# API Inquiry v0.3.1

## New Features

- Added a Codex provider that queries Codex quota through the ChatGPT/Codex session quota endpoint.
- Codex first auto-reads local Codex login state (`$CODEX_HOME/auth.json` or `~/.codex/auth.json`), so users do not need to enter an OpenAI Platform API key in the Console.
- When Codex is the Primary Provider, the menu bar keeps the compact image-label design and shows the ChatGPT/GPT mark with `5h xx%` remaining quota.
- The Codex detail panel shows both 5h and Week remaining quota windows with reset times.
- Console Home shows the current Codex plan, such as Free, Plus, Pro 5x, Pro 20x, Business, or Enterprise/Edu.

## Known Limits

- No Apple notarization.
- Codex quota checks depend on a ChatGPT/Codex session backend that may change with Codex itself.
- Only built-in providers are supported; arbitrary custom endpoints are not supported.
