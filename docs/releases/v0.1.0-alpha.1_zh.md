# API Inquiry v0.1.0-alpha.1

API Inquiry 的 alpha GitHub DMG 版本。API Inquiry 是一个原生 macOS 菜单栏应用，用于查询 DeepSeek API 账号余额。

这是一个早期测试版本。余额查询、Keychain 存储、自动刷新、手动刷新、AutoStart 和菜单栏 UI 已可用，但还没有进行 Apple notarization 公证，也没有经过大范围兼容性验证。

## 安装

1. 下载 `API-Inquiry-alpha.dmg`。
2. 打开 DMG。
3. 将 `API Inquiry.app` 拖入 `Applications`。
4. 从 Applications 启动 API Inquiry。

## 首次启动提示

本 alpha 版本采用免费发布路线。应用已经进行 ad-hoc 签名，但没有 Apple notarization 公证，因此 macOS 首次启动时可能提示“无法验证开发者”。

如果 macOS 阻止打开：

1. 右键点击 `API Inquiry.app`。
2. 选择 `Open`。
3. 在系统提示中再次确认 `Open`。

也可以进入 `System Settings > Privacy & Security`，允许该应用打开。

## 已包含

- 通过 DeepSeek 官方余额 API 查询余额。
- API Key 安全存储在 macOS Keychain。
- 每 5 分钟自动刷新和手动刷新。
- 原生 macOS 菜单栏 UI。
- AutoStart 开机自启开关。
- 自定义应用图标。
- 最近更新时间跟随系统 12 小时制 / 24 小时制。

## 已知限制

- 未进行 Apple notarization 公证。
- 首次启动可能需要在 macOS 中手动允许打开。
- 没有详细用量图表。
- 没有本地 DeepSeek 用量控制台。
- 当前 alpha 版本界面中只显示 DeepSeek 一个供应商。
