# API Inquiry v0.3.4

## Polish

- The primary number in the menu details panel now uses health colors so the current provider's quota state is easier to scan at a glance.
- DeepSeek-style balance providers show green, amber, or red based on the remaining balance amount.
- Zhipu GLM Coding Plan-style used-plan providers show green, amber, or red based on the used percentage.
- OpenAI/Codex remaining quota rows show green, amber, or red based on each remaining quota window.
- The warning color has been tuned to a brighter amber tone for clearer attention without changing the existing success and critical tones.
- DeepSeek and Zhipu primary detail numbers now share a stable 42 pt size, with symbols and units kept proportionally smaller.
- The app icon's lower-right usage bars now echo the critical, warning, and healthy color system in a softer icon-friendly treatment.
- The DMG install window now uses a larger drag-to-Applications layout with a clearer arrow guide and centered app targets.

## Fixes

- Fixed the menu details panel showing different primary number sizes when the app first opened versus after changing the primary provider in Console.
