# API Inquiry v0.3.0

## 中文

### 新增功能

- 新增内置多供应商目录，第一批支持 DeepSeek 和智谱 GLM Coding Plan。
- 新增智谱 GLM Coding Plan 用量查询，支持查看 plan 窗口、使用百分比和下次重置时间。
- 菜单栏改为只显示 Primary Provider：DeepSeek 显示余额，智谱显示类似 `5h 17%` 的 plan 用量。
- 菜单栏详情页顶部突出展示 Primary Provider，并以紧凑行展示其他已添加供应商。
- 菜单栏详情页中，智谱展示 `Resets`，Console Home 展示 `Plan Next Resets`。
- Console 支持添加供应商、按供应商保存/替换/删除 API Key，并设置菜单栏 Primary Provider。
- 每个供应商使用独立 Keychain account，刷新状态、最后快照和错误状态互不影响。

### 优化改进

- 优化 Home 和 API 页布局，使供应商卡片、指标列、操作按钮和供应商 logo 的高度与对齐更一致。
- 菜单栏详情页刷新按钮现在会刷新所有已添加供应商，而不只刷新 Primary Provider。
- 打磨 DeepSeek 和智谱的详情展示，包括货币字号、plan 用量字号，以及跟随系统 12/24 小时制的时间格式。

### 验证下载文件

```bash
shasum -a 256 -c API-Inquiry-v0.3.0.dmg.sha256
```

### 已知限制

- 未进行 Apple notarization 公证。
- 当前只支持内置供应商，不支持任意自定义 endpoint。
- 不包含历史用量导入、图表或跨供应商聚合统计。

## English

### New Features

- Added a built-in multi-provider catalog with DeepSeek and Zhipu GLM Coding Plan as the first providers.
- Added Zhipu GLM Coding Plan usage checks, including plan window, usage percentage, and next reset time.
- Changed the menu bar to show only the Primary Provider: DeepSeek shows balance, while Zhipu shows plan usage such as `5h 17%`.
- Updated the menu bar details panel to feature the Primary Provider at the top and show other added providers as compact rows.
- Added Zhipu reset visibility: `Resets` in the menu details panel and `Plan Next Resets` in Console Home.
- Added Console management for adding providers, saving/replacing/deleting provider-specific API keys, and selecting the Primary Provider shown in the menu bar.
- Kept each provider isolated with its own Keychain account, refresh state, last snapshot, and error state.

### Improvements

- Refined the Home and API layouts so provider cards, metric columns, actions, and provider logos align consistently.
- Updated the menu details refresh action to refresh every added provider, not only the Primary Provider.
- Polished DeepSeek and Zhipu detail formatting, including currency typography, plan usage typography, and system 12-hour/24-hour time formatting.

### Verify Download

```bash
shasum -a 256 -c API-Inquiry-v0.3.0.dmg.sha256
```

### Known Limits

- No Apple notarization.
- Only built-in providers are supported; arbitrary custom endpoints are not supported.
- Historical usage import, charts, and cross-provider aggregate statistics are not included.
