# API Inquiry v0.3.9

## 新增功能

- Console Home 的 OpenAI/Codex 供应商卡片现在展示 Codex 手动重置额度，包含可用重置卡数量和最近到期时间。
- 手动重置额度可以在 Console Home 指标旁单独刷新，不加入菜单栏主刷新链路。

## 应用优化

- 手动重置额度查询使用独立低频缓存，不阻塞常规 5 分钟供应商刷新，也不阻塞菜单栏手动刷新。
- 菜单栏刷新和手动重置刷新统一使用轻量的查询中、成功、失败反馈。
- OpenAI/Codex Home 卡片在现有紧凑供应商布局内展示额度、手动重置和更新时间，避免额外增加信息层级。

## 问题修复

- 修复反复点击菜单栏刷新后，刷新图标看起来越来越快的问题。

## 下载校验

```bash
shasum -a 256 -c API-Inquiry-v0.3.9.dmg.sha256
```

## 已知限制

- Apple notarization 尚未启用。
- 手动重置额度依赖当前本机 Codex/ChatGPT 凭证；如果凭证过期，需要先刷新登录态再查询。
