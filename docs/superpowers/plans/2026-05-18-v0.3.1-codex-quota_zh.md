# API Inquiry v0.3.1 Codex 额度实施计划

> **给 agentic workers：** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标：** 新增 Codex provider，自动读取本机 Codex 登录态，通过 ChatGPT/Codex 会话型 quota 接口展示 5h 和 Week 剩余额度，并在 Console 展示当前 plan。

**架构：** 在现有 `ProviderSnapshot` 体系中新增 `quotaUsage` 分支，Codex credential source 优先读取 `$CODEX_HOME/auth.json` 或 `~/.codex/auth.json`，缺失时才回退 Keychain；Codex provider 负责凭证解析和 `/backend-api/wham/usage` 响应解析。菜单栏、详情页和 Console 继续通过 ViewModel/Formatter 派生 UI 文案，避免 SwiftUI 直接理解网络 schema。

**技术栈：** Swift 5.9、SwiftUI、AppKit、URLSession-backed `HTTPClient`、macOS Keychain、UserDefaults、Swift Package Manager、本地 `APIInquiryCoreTestsRunner`。
