# API Inquiry v0.2.0 供应商控制台实施计划

> **给 agentic workers：** v0.2.0 范围已在 2026-05-17 调整。为了保持应用小巧，历史用量导入已移除。只实现本文描述的供应商控制台。

**目标：** 构建轻量本地控制台，包含 Home 和 API 两页，用于查看供应商状态和管理 API Key。

**架构：** 继续使用现有余额 controller 和 credential store。将 `UsageConsoleViewModel` 职责收敛到供应商摘要和 API Key 操作。从 active app 中移除用量导入、解析、存储和展示代码。

**技术栈：** Swift 5.9、SwiftUI、AppKit、macOS Keychain、Swift Package Manager、本地 `APIInquiryCoreTestsRunner`。
