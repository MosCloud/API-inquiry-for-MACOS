# API Inquiry v0.2.0 用量控制台实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标：** 在 `v0.2.0` 分支中实现本地用量控制台、DeepSeek Usage CSV 导入、本地用量汇总，并将 API Key 管理从菜单栏面板迁移到控制台中。

**架构：** `APIInquiryCore` 新增纯 Swift 用量模型、CSV parser、JSON store 和控制台 view model；`APIInquiryApp` 新增可复用控制台窗口与 SwiftUI 视图。菜单栏继续复用同一个余额 controller，但只保留状态和入口，不再承载完整 API Key 表单。

**技术栈：** Swift 5.9、SwiftUI、AppKit、Foundation CSV/JSON 处理、macOS Keychain、Swift Package Manager、本地 `APIInquiryCoreTestsRunner`。

---
