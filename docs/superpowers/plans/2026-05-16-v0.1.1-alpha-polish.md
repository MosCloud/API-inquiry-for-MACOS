# API Inquiry v0.1.1 Alpha Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.1.1 as a focused alpha polish release with clearer first-run setup, actionable error recovery, safer API key management feedback, fast in-progress app launching, and more trustworthy GitHub DMG release artifacts.

**Architecture:** Add a small failure classification layer in `APIInquiryCore`, expose derived UI actions from `MenuBarBalanceViewModel`, and keep `MenuBarContentView` declarative. Keep development validation local-first through a quick run script, then reserve DMG packaging, checksum generation, and release notes for release candidate validation.

**Tech Stack:** Swift 5.9, SwiftUI `MenuBarExtra`, macOS Keychain, Bash packaging scripts, ad-hoc `codesign`, `hdiutil`, SHA-256 checksum via `shasum`.

---

## Scope Check

This plan intentionally keeps v0.1.1 small. It does not build the local DeepSeek console, usage charts, multi-provider UI, full localization, Developer ID signing, or Apple notarization.

```swift
private static func failureKind(for error: Error) -> BalanceFailureKind {
    if let providerError = error as? BalanceProviderError {
        return providerError.failureKind
    }

    if let urlError = error as? URLError {
        switch urlError.code {
        case .notConnectedToInternet, .networkConnectionLost, .cannotFindHost, .cannotConnectToHost, .timedOut:
            return .networkUnavailable
        default:
            return .unknown
        }
    }

    return .unknown
}
```

```bash
INFO_PLIST="$VERIFY_MOUNT/$APP_NAME/Contents/Info.plist"

    echo "Executable missing from $DMG_NAME." >&2
    exit 1
fi

BUNDLE_ID=$(/usr/libexec/PlistBuddy -c "Print CFBundleIdentifier" "$INFO_PLIST")
APP_VERSION_IN_PLIST=$(/usr/libexec/PlistBuddy -c "Print CFBundleShortVersionString" "$INFO_PLIST")
BUILD_NUMBER_IN_PLIST=$(/usr/libexec/PlistBuddy -c "Print CFBundleVersion" "$INFO_PLIST")

if [ "$BUNDLE_ID" != "com.api-inquiry.APIInquiry" ]; then
    echo "Unexpected bundle id: $BUNDLE_ID" >&2
    exit 1
fi

if [ "$APP_VERSION_IN_PLIST" != "$APP_VERSION" ]; then
    echo "Unexpected app version: $APP_VERSION_IN_PLIST" >&2
    exit 1
fi

if [ "$BUILD_NUMBER_IN_PLIST" != "$BUILD_NUMBER" ]; then
    echo "Unexpected build number: $BUILD_NUMBER_IN_PLIST" >&2

fi
```
