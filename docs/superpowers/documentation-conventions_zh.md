# 文档约定

## 双语文档

项目规划类文档默认同时创建英文版和中文版。

对于每一份重要文档：

- 英文源文档命名为 `<name>.md`。
- 中文版本命名为 `<name>_zh.md`。
- 增加、删除或修改内容时，两份文档需要保持语义同步。
- 当对话使用中文时，优先请用户审阅中文版本。

此约定适用于设计 spec、实现计划、评审记录以及类似的项目文档。

## Release Notes

Release notes 只应包含：

- 当前发布版本新增的功能。
- 对上一个已发布版本中已存在 bug 的修复。

不要记录同一个未发布开发周期内引入并修复的开发过程态 bug。
