# API Inquiry v0.3.10 手动重置详情与时区自适应设计

## 摘要

v0.3.10 在 v0.3.9 的 Codex 手动重置摘要基础上继续收窄边界：修复代码审阅中“应优先处理”的四个问题，为 Console Home 的 OpenAI/Codex 卡片增加手动重置详情入口，并统一应用内时间展示的时区策略。

本版本不改变 Menubar 的信息架构，不把手动重置加入 Menubar 刷新链路，不实现执行手动重置的动作。

## 目标

- 修复 README 测试期望数与功能描述漂移。
- 让智谱用量百分比能解析小数，避免接口返回 `17.4` 时整次查询失败。
- 让 Codex `reset_at` 兼容 Unix 秒、Unix 毫秒、数字字符串和 ISO-8601 字符串。
- 删除 OpenAI/Codex 供应商后清空手动重置缓存，避免重新添加时短暂显示旧账号数据。
- 点击 Console Home OpenAI 卡片的手动重置摘要值时，弹出手动重置详情页。
- 详情页展示所有手动重置卡的发放时间和过期时间。
- 手动重置摘要和详情页时间展示跟随系统时区。

## 非目标

- 不修改 Menubar 展示内容。
- 不让 Menubar 手动刷新等待或触发 reset credits 查询。
- 不增加 Console Home 全局刷新按钮。
- 不改变 v0.3.9 的独立 1 小时缓存、局部刷新和成功/失败动效。
- 不保存或展示 token、refresh token、cookie、完整账号 ID 或原始响应体。
- 不新增历史趋势、到期提醒、手动重置执行动作或持久化 reset credits 存储。

## 手动重置详情入口

入口放在 OpenAI/Codex 卡片现有 `手动重置` metric 的值区域：

```text
手动重置        ↻
3 张 · 07/18 到期
```

- 值区域可点击，点击打开详情页。
- 标题右侧的 icon-only 刷新按钮继续只刷新 reset credits。
- 刷新按钮和可点击值区域不能互相覆盖，交互 hit area 分离。
- 未配置 Codex 或没有手动重置 metric 时不提供详情入口。

不把整块 metric 包成按钮，避免与局部刷新按钮形成嵌套交互。

## 详情页 UI

详情页使用附着在 Console 主窗口上的 sheet，不打开独立外部窗口。

视觉要求：

- 风格与主 Console 统一，使用现有深色材质、圆角尺度、低对比边界和紧凑排版。
- 不使用新的装饰风格、重型卡片、渐变背景或与主界面不一致的视觉语言。
- 表格采用轻量 header、行分隔线和等宽时间文本，保持信息可扫读。
- 空状态只展示简洁文案，例如 `暂无手动重置记录`。
- 关闭按钮使用低强调按钮或系统语义，不抢占内容层级。

详情页内容：

```text
手动重置详情

发放时间                            过期时间
2026-06-18 08:35:47 UTC+08:00       2026-07-18 08:35:47 UTC+08:00
2026-06-27 07:44:20 UTC+08:00       2026-07-27 07:44:20 UTC+08:00
```

表头只展示字段名，不追加固定时区文案。每行时间值按系统本地时区转换，并在时间文本末尾展示该日期对应的 `UTC±HH:MM` 偏移。

详情列表展示所有返回的手动重置卡，不只展示可用卡。排序按过期时间升序，缺失过期时间的记录放在最后。

## 时区策略

应用内用户可见时间遵循系统本地时区：

- `LastRefreshTimeFormatter` 继续使用可注入的 locale 和 time zone。
- 手动重置摘要不再使用写死的 `Asia/Shanghai` calendar。
- 手动重置详情使用与 Console 现有时间一致的时间上下文。
- 内部解析仍以接口返回时间为准；只在展示层做本地时区转换。

这意味着同一个 UTC 时间在不同时区可能显示不同的日期，这是正确行为。

## 测试

核心测试覆盖：

- README 中测试期望数与功能描述更新。
- 智谱小数百分比解析为 `Decimal`，且可用性判断不被整数化影响。
- Codex `reset_at` 解析 Unix 毫秒和 ISO-8601 字符串。
- 删除 Codex provider 后清空手动重置状态和缓存。
- 手动重置摘要在 `Asia/Shanghai` 和 `America/Los_Angeles` 下显示不同本地日期。
- 详情 display model 输出所有卡的发放/过期时间，并带系统时区后缀。
- 中文和英文新增文案都有测试覆盖。

App/UI 构建验证覆盖：

- Console Home 的手动重置值区域可打开 sheet。
- 局部刷新按钮仍可单独点击，且不与标题和值重叠。
- 详情页视觉保持主 Console 风格。
